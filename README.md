# Commands

Command | Purpose
--- | --
`^` | Set current pointer to corrosponding input
`?` | Echo current pointer
`!` | Echo current pointer's data
`*` | Exit prematurely
`+` | Add 1 to pointer selected
`-` | Remove 1 from pointer selected
`>` | Move to right pointer
`<` | Move to left pointer
